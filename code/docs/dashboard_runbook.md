# Dashboard Runbook (Big Screen)

This repository includes a lightweight, dependency-free, “big screen” dashboard under `web/`.
It reads the latest `docs/risk_report.json` generated by the pipeline and renders:
- global KPIs
- risk level distribution
- top signal dimensions
- a sortable/filterable repo list + evidence-chain detail panel

## 1) Generate data

Make sure you already generated `docs/risk_report.json`:

```bash
python scripts/build_risk_explanations.py --sources configs/sources.yaml --model configs/model.yaml --signals configs/signals.yaml --model-type baseline --replace
python scripts/export_riskreports.py --sources configs/sources.yaml --model-type baseline
```

## 2) Start local server

From repo root:

```bash
python -m http.server 8000
```

## 3) Open the dashboard

Open:
- `http://localhost:8000/web/`

The dashboard fetches:
- `../docs/risk_report.json` (relative to `web/`)
